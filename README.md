# AI WhatsApp Automation Workflow

This project is an AI-powered WhatsApp automation workflow built using n8n.

## Overview
The workflow automates WhatsApp message handling and logs chat data into Google Sheets.  
It helps in maintaining structured datasets and reduces manual data entry.

## Features
- Automated WhatsApp message responses
- AI-assisted conversation handling
- Chat data logging into Google Sheets
- Structured and clean data storage
- Reduced manual data handling

## Tools & Technologies
- n8n (Workflow Automation)
- AI Chat Model
- Google Sheets
- No-code / Low-code Automation

## What I Learned
- Building automation workflows using n8n
- Handling and structuring real-world data
- Using AI tools as an assistant for efficiency
- Data logging and basic data management

## Usage
This workflow was created for learning and practical experimentation.  
Sensitive credentials and configuration details have been removed for security reasons.
